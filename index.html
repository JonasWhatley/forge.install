How to install Forge mods

Author: Jonas Whatley

11/24/20 9:00 pm

Download Java

In order to install anything that resembles a mod, you need to install Java. Java is the coding language that minecraft was built on and as a result, the mods are too. You can download java here.

Download Forge

Forge is the "mod loader" for Minecraft. If you want to install mods, you must have foge, otherwise Minecraft will not know what to do with them. You can download Forge here. Once you downloaded your Forge version, open the jar file by double clicking on it and install.

Create the "mods" folder

Minecraft and Forge need to know where your mods are. Open Run on your computer by pressing the Windows key and R at the same time. You will see a window open, in this window, type in "%appdata%". Press enter. Now you will see a bunch of folders. Clock on the one that says ".minecraft". In here, create a folder named "mods" and put all of your mods in there.

Create the "mods" folder

We're almost there.. We have created a drive link to all of the mods that you will need to play on our server. If you are having trouble installing them, read the above instructions closely. You can download the mods here.

Put the mods in your mod folder

Once the mods are finished downloading, You are ready put them into the mods folder. Go back to the "mods" folder in the ".minecraft" folder in your computer. If you can't find it, simply follow the 3rd instruction above.

Wait.. Are we really done?

Yes! All you have to do is allocate a bit more ram into minecraft. Open your minecraft launcher and head over to the "installations" tab at the top. In here, click on the "Forge" option and click on "more options". You should see a "JVM ARGUMENTS" section. Change the "-Xmx2G" to "-Xmx4G" and you hould be good to go. Enjoy!
